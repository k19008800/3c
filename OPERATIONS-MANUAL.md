## 快速开始

### 登录系统
1. 访问 `https://unmisa.com/login`
2. 输入管理员邮箱和密码
3. 完成两步验证（如已启用）
4. 进入管理后台

### 权限说明
| 角色 | 权限范围 |
|------|----------|
| **超级管理员** | 全部功能 |
| **运营管理员** | 用户管理、数据分析、内容运营 |
| **财务管理员** | 财务结算、充值审核、提现审核 |
| **系统管理员** | 系统配置、安全风控、运维监控 |

## 总览看板

### 📊 管理仪表盘 (`/console/admin`)

**功能定位**: 运营人员的每日工作台，全局业务健康度概览

**核心功能**:
1. **实时业务指标**
   - 在线通道数：当前活跃的供应商通道
   - 今日调用量：今日API调用总次数
   - 今日Token消耗：今日Token使用总量
   - 今日营收：今日收入金额
   - 异常告警：待处理的告警数量

2. **趋势分析图表**
   - 总调用量趋势（7天/30天）
   - Token消耗趋势
   - 营收趋势
   - 活跃用户数趋势

3. **快捷操作**
   - 快速审核充值订单
   - 快速处理提现申请
   - 快速查看异常告警
   - 快速添加新用户

4. **数据表格**
   - Top消费用户排行
   - 最近充值记录
   - 待办事项队列

**使用场景**:
- 每日上班查看业务概况
- 快速发现异常和问题
- 处理待审核的业务单据

**操作技巧**:
- 点击数据卡片可跳转到详情页
- 图表支持时间范围切换
- 待办事项可一键处理

---

### 📈 企业数据分析 (`/console/admin/enterprise-analysis`)

**功能定位**: 企业客户分析，识别高价值客户和风险客户

**核心功能**:
1. **企业统计概览**
   - 企业总数：注册企业数量
   - 总余额：所有企业账户余额总和
   - 月活跃企业：本月有消费的企业数
   - 低余额企业：余额低于¥10的企业数

2. **消费统计**
   - 本月企业总消费
   - 本月Token消耗量
   - 消费环比增长率

3. **趋势分析**
   - 企业整体消费趋势（30天）
   - 新增企业数量趋势
   - 企业活跃度分布

4. **企业排行**
   - Top 10消费企业
   - Top 10活跃企业
   - 低余额预警列表

**使用场景**:
- 识别高价值企业客户
- 发现流失风险客户
- 制定企业运营策略

**操作技巧**:
- 搜索框支持企业名称/邮箱搜索
- 点击企业名称可查看详情
- 低余额企业可一键发送提醒

## 常见问题

### Q1: 如何快速审核充值订单？
**A**: 在财务工作台点击待审核充值订单，查看支付凭证，核对金额和用户信息，通过或拒绝审核。

### Q2: 如何处理用户余额异常？
**A**: 在用户管理页面找到用户，查看余额流水记录，手动调整余额并填写调整原因。

### Q3: 如何添加新的供应商？
**A**: 在供应商管理页面点击添加供应商，填写API配置信息，测试连通性，同步模型列表。

### Q4: 如何配置代理商佣金？
**A**: 在代理商管理页面找到代理商，编辑佣金比例，设置推广码，保存配置。

### Q5: 如何查看系统异常？
**A**: 在系统健康页面查看服务状态，在安全事件页面查看异常事件，在熔断看板查看熔断状态。

### Q6: 如何导出数据报表？
**A**: 在各统计页面点击导出按钮，选择导出格式（CSV/Excel），选择时间范围，下载报表。
