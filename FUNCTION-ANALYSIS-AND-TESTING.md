# 3cloud 项目全局功能推演与测试报告

## 一、业务思路推演（AI Token 聚合平台核心功能）

### 1.1 核心业务流
```
用户调用 → 鉴权认证 → 余额检查 → 模型路由 → 供应商转发 → 响应回传 → 精确计费 → 日志记录
```

### 1.2 必须的全局管理功能

| 功能域 | 子功能 | 业务价值 |
|--------|--------|----------|
| **资源管理** | 供应商管理、模型管理、API Key 管理、Key 资源池 | 确保上游供应稳定，成本可控 |
| **财务管理** | 实时计费、余额管理、充值审核、提现审核、结算对账、发票管理 | 资金安全，财务透明，合规运营 |
| **用户运营** | 用户管理、实名审核、配额管理、角色权限、代理商管理 | 用户分级，风险控制，增长驱动 |
| **安全风控** | 登录安全、操作审计、限流熔断、敏感词检测、安全事件 | 系统稳定，数据安全，合规运营 |
| **数据分析** | 仪表盘、统计报表、成本分析、使用趋势、盈利分析 | 决策支持，优化定价，发现机会 |
| **系统运维** | 系统健康、配置管理、邮件模板、页面内容、公告管理 | 系统稳定，运营效率，品牌一致 |

## 三、现有管理功能描述说明

### 3.1 仪表盘与概览模块

#### 📊 Dashboard（控制台首页）
**功能说明**：运营人员每日必看的数据总览，展示核心业务指标实时状态。

**核心指标**：
.#### 实时数据卡片
- 今日 Token 消耗量（千为单位）
- 今日 API 调用次数
- 今日充值金额
- 在线用户数
- 可用供应商数
- 活跃代理商数

.#### 快速操作入口
- 用户审核（实名认证待处理）
- 财务审核（充值/提现待处理）
- 安全警报（异常登录/调用）

**使用场景**：
- 运营人员晨会查看昨日数据
- 实时监控业务健康度
- 快速处理待办事项

#### 📈 Stats（统计分析）
**功能说明**：多维度的数据统计分析，支持自定义时间范围和指标对比。

**分析维度**：
- 时间趋势：日/周/月/年数据对比
- 用户维度：新老用户贡献分析
- 模型维度：各模型使用量和收入
- 代理商维度：渠道贡献和佣金

**使用场景**：
- 月度经营分析报告
- 产品定价策略调整
- 市场推广效果评估

#### 🔍 Trends（趋势洞察）
**功能说明**：深入的趋势分析和异常检测，识别业务模式和风险点。

**洞察能力**：
- 使用高峰期识别
- 异常消费模式检测
- 用户行为聚类分析
- 季节性趋势预测

**使用场景**：
- 容量规划和服务扩容
- 风险用户早期识别
- 产品功能优化方向

### 3.2 用户运营模块

#### 👤 Users（用户管理）
**功能说明**：全平台用户生命周期管理，从注册到注销的全流程管控。

**管理能力**：
- 用户信息查看和编辑
- 实名认证审核
- 账户状态管理（启用/禁用/冻结）
- 余额和配额调整
- 操作日志追溯

**使用场景**：
- 新用户人工审核（高风险行业）
- 违规用户处理
- 大客户特殊支持
- 用户投诉处理

#### 🏦 Agents（代理商管理）
**功能说明**：渠道合作伙伴体系管理，支持多层代理和佣金结算。

**管理功能**：
- 代理商入驻审核
- 佣金比例配置
- 下级用户管理
- 提现申请审核
- 业绩统计报表

**使用场景**：
- 渠道招商和签约
- 代理商绩效考核
- 佣金结算和发放
- 渠道政策调整

#### 🔐 Roles（角色权限）
**功能说明**：基于RBAC的精细化权限控制系统，支持自定义角色和权限组合。

**权限维度**：
- 功能权限：页面访问和操作
- 数据权限：数据范围隔离
- 操作权限：增删改查控制

**使用场景**：
- 新员工权限分配
- 临时权限授予（如审计人员）
- 权限最小化原则实施

### 3.3 资源管理模块

#### 🤖 Models（模型管理）
**功能说明**：AI模型全生命周期管理，从接入到下线全流程管控。

**管理内容**：
- 模型基本信息（名称、供应商、上下文长度）
- 定价策略（输入/输出价格、倍数）
- 启用状态和排序
- 使用统计和成本分析

**使用场景**：
- 新模型上线流程
- 价格策略调整
- 模型性能监控
- 下线模型归档

#### 🏢 Vendors（供应商管理）
**功能说明**：上游AI供应商管理，确保供应链稳定和成本优化。

**管理维度**：
- 供应商基本信息
- API Key 资源池管理
- 连通性测试和监控
- 合作关系管理

**使用场景**：
- 新供应商接入评估
- 供应商性能监控
- 故障切换和容灾
- 采购谈判支持

#### 🔑 API Keys（API密钥管理）
**功能说明**：用户API密钥的创建、管理和安全控制。

**安全特性**：
- 密钥哈希存储（前端仅显示前缀）
- 使用量统计和告警
- 密钥轮换和失效
- 操作审计日志

**使用场景**：
- 用户密钥泄露处理
- 企业密钥批量管理
- 密钥使用异常检测

### 3.4 财务管理模块

#### 💰 Finance Dashboard（财务总览）
**功能说明**：财务健康状况全景视图，现金流和盈利能力监控。

**监控指标**：
- 实时平台总余额
- 今日收入/支出
- 待处理充值/提现
- 逾期应收账款

**使用场景**：
- 每日资金头寸管理
- 现金流预警
- 财务报告数据源

#### 💳 Recharge Orders（充值订单）
**功能说明**：用户充值流程管理，支持人工审核和自动化处理。

**处理流程**：
1. 用户提交充值申请（上传凭证）
2. 财务人员审核银行信息
3. 确认到账后增加余额
4. 通知用户充值成功

**审核要点**：
- 银行账户信息匹配
- 转账金额和时间一致
- 凭证真实性验证

#### 🏧 Withdraws（提现申请）
**功能说明**：用户和代理商提现申请处理，严格的双审核机制。

**安全机制**：
- 初级审核：业务审核（余额、合规性）
- 二级审核：财务审核（银行信息、金额）
- 大额提现：额外安全验证

**使用场景**：
- 常规提现处理
- 大额提现风险控制
- 节假日提现安排

#### 📊 Settlement（结算对账）
**功能说明**：周期性结算单管理，确保平台和供应商对账准确。

**结算流程**：
1. 生成结算周期账单
2. 锁定结算单（防止数据变更）
3. CSV导出对账明细
4. 资金划转和确认

**对账维度**：
- 代理商佣金结算
- 供应商应付账款
- 平台毛利润计算

### 3.5 安全风控模块

#### 🛡️ Security Events（安全事件）
**功能说明**：全平台安全事件监控和响应，实时告警和处置。

**监控范围**：
- 异常登录行为
- API调用异常模式
- 敏感操作审计
- 系统资源异常

**响应流程**：
1. 事件检测和告警
2. 初步风险评估
3. 处置措施执行
4. 事后分析和改进

#### 🔒 Security Bans（封禁管理）
**功能说明**：风险用户和IP的封禁管理，支持自动和手动封禁。

**封禁策略**：
- 自动封禁：基于规则引擎
- 手动封禁：运营人员判断
- 临时封禁：调查期间限制
- 永久封禁：严重违规处理

**使用场景**：
- 恶意爬虫防护
- 欺诈行为处理
- DDOS攻击防护

#### 📝 Audit Logs（审计日志）
**功能说明**：关键操作的全链路审计，满足合规要求和事故调查。

**审计范围**：
- 管理员操作日志
- 财务相关操作
- 用户敏感操作
- 系统配置变更

**合规价值**：
- 满足GDPR/数据安全法要求
- 内部控制和责任追溯
- 事故根本原因分析

### 3.6 系统运维模块

#### ⚙️ System Health（系统健康）
**功能说明**：基础设施和业务系统健康状态监控。

**监控指标**：
- 数据库连接和性能
- Redis缓存状态
- 外部API连通性
- 服务响应时间
- 错误率和异常

**告警机制**：
- 分级告警（警告/严重/灾难）
- 多渠道通知（邮件/短信/钉钉）
- 自动恢复尝试

#### 🔧 Site Settings（站点配置）
**功能说明**：系统全局配置管理，支持热更新和版本控制。

**配置类型**：
- 业务参数（费率、限制）
- 功能开关（A/B测试）
- 显示配置（主题、文案）
- 第三方集成

**管理原则**：
- 配置版本化和回滚
- 变更审批流程
- 影响评估和测试

#### 📧 Email Templates（邮件模板）
**功能说明**：系统邮件模板管理，支持多语言和变量替换。

**模板类型**：
-  transactional（交易通知）
-  marketing（营销推广）
-  security（安全通知）
-  system（系统通知）

**管理功能**：
- 富文本编辑和预览
- 变量语法验证
- 发送测试和统计
