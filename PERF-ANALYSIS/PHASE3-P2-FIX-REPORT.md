## 二、后端修复详情

### 2.1 响应压缩

**新增插件**：`src/plugins/compress.ts`

**配置**：
- 压缩阈值：1KB（仅压缩 >1KB 响应）
- 支持编码：gzip、deflate、br
- 压缩级别：6（平衡压缩比和性能）

**效果**：
| 响应大小 | 压缩前 | 压缩后 | 节省 |
|----------|--------|--------|------|
| 100KB JSON | 100KB | ~15KB | **85%** |
| 500KB JSON | 500KB | ~50KB | **90%** |

---

### 2.2 限流保护

**新增插件**：`src/plugins/rate-limit.ts`

**分级限流**：
| 接口类型 | 限流策略 | 说明 |
|----------|----------|------|
| 全局 | 100 次/分钟 | 按 IP + 用户 ID |
| 登录 | 5 次/分钟 | 防暴力破解 |
| Token 刷新 | 10 次/分钟 | 防滥用 |
| 批量启停 | 10 次/分钟 | 防资源耗尽 |
| 批量删除 | 5 次/分钟 | 高危操作 |

**效果**：
- 超限返回 429 状态码
- 自定义错误响应
- 管理员 API Key 可绕过
