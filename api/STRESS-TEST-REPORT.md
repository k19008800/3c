## 六、限流分析

### 问题
测试期间大量请求返回 **429 Too Many Requests**，原因：
1. **全局限流配置**: 系统默认限流规则阻止了高频并发
2. **并发度过高**: 20 并发 + 100ms 间隔触发了限流阈值

### 建议
1. 压力测试前临时调高限流阈值
2. 或使用多个 API Key 分散限流计数
3. 或在测试环境禁用限流（仅限测试）
